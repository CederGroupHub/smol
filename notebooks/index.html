# Statistical Mechanics on Lattices Examples#

The following jupyter notebook examples are provided to get you started and up to speed with the basic functionality of the package.

We try to keep notebooks updated to run the latest version of the package, and additional notebooks are usually added when important new functionality is added.

We also encourage users to contribute their own examples of useful workflows and calculations using the package. For further information refer to the this section of the documentation.

Notebook examples#

Basic examples#

  • Creating a basic cluster expansion: Shows how to set up, create and fit a basic cluster expansion.

  • Creating a cluster expansion with electrostatics: Shows how to set up, create and fit a basic cluster expansion of an ionic system with an additional Ewald electrostatic term.

  • Visualizing clusters: Demonstrates how to use the crystal-toolkit package to visualize structures and clusters.

  • Running canonical Monte Carlo: Shows an example of running simple canonical Monte Carlo sampling using a cluster expansion.

  • Running semigrand canonical Monte Carlo: Shows an example of running simple semigrand canonical Monte Carlo sampling using a cluster expansion.

  • Running charge neutral semigrand canonical Monte Carlo: An example of running charge neutral semigrand canonical Monte Carlo sampling using a cluster expansion with ionic species.

  • Wang-Landau sampling of an FCC Ising model: An example of using Wang-Landau sampling to estimate the DOS of an anti-ferromagnetic FCC Ising model.

  • Preparing cluster expansion training data: Shows how to modify training data and use weights for additional cluster expansion fitting control.

Advanced examples#

  • Centering fitting data in a piece-wise fit: An example showing how to center fitting data when doing a “piece-wise” fit with electrostatics.

  • Adding structures to a StructureWrangler in parallel: Adding structures to fit a cluster expansion can be time-consuming one way to improve performance is to add them in parallel.

  • Simulated annealing with point electrostatics: This example shows how to run simulated annealing using a point electrostatic potential which can be useful to obtain initial training structures for ionic systems.

  • Li-Mn-O DRX cluster expansion and sampling: Demonstrates how to use the crystal-toolkit package to visualize structures and clusters.
